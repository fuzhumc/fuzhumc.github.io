常用 Hexo 命令

命令 功能
hexo new post 标题 创建一篇新文章
hexo clean 清除生成的文件
hexo generate 生成静态文件
hexo server 启动本地服务器预览博客
hexo deploy 部署博客到 GitHub Pages
